# MedServicePrice.kz

## Описание проекта

MedServicePrice.kz — веб-приложение для поиска и сравнения стоимости медицинских услуг в клиниках Казахстана.

Основная задача проекта — упростить поиск медицинских услуг для пациентов. Вместо самостоятельного просмотра сайтов различных клиник пользователь может воспользоваться единым сервисом, который позволяет найти нужную услугу, сравнить стоимость в разных медицинских центрах и выбрать наиболее подходящий вариант.

Проект разработан в рамках MedTech Hackathon.

---

## Основной функционал

В текущей версии реализованы следующие возможности:

* поиск медицинских услуг;
* сравнение стоимости услуг между клиниками;
* фильтрация по городу и категории;
* сортировка результатов по цене;
* отображение клиник на интерактивной карте;
* нормализация различных названий одной и той же услуги (например, «ОАК», «Общий анализ крови», «CBC»);
* AI-помощник, который анализирует описание симптомов пользователя и предлагает медицинские услуги, которые могут быть полезны в данной ситуации.

---

## Принцип работы

После запуска приложения база данных автоматически заполняется информацией о клиниках и медицинских услугах из заранее подготовленного списка источников.

Пользователь может:

1. найти интересующую медицинскую услугу;
2. посмотреть стоимость в различных клиниках;
3. открыть расположение клиники на карте;
4. воспользоваться AI-помощником для получения рекомендаций по необходимым обследованиям или консультациям специалистов.

## Запуск проекта

Создать виртуальное окружение

```bash
python -m venv .venv
```

Активировать окружение

Windows

```bash
.venv\Scripts\activate
```

Установить зависимости

```bash
pip install -r requirements.txt
```

Создать файл `.env`

```text
OPENAI_API_KEY=ваш_API_ключ
```

Запустить сервер

```bash
cd backend

python -m uvicorn main:app --reload --port 8000
```

После запуска приложение будет доступно по адресу

```text
http://localhost:8000
```

## Работа с AI-помощником

Пользователь может описать симптомы в свободной форме.

Например:

* болит живот;
* боль в груди;
* высокая температура несколько дней;
* болит горло.

На основе запроса AI формирует список рекомендуемых медицинских услуг, после чего система автоматически выполняет поиск этих услуг среди клиник, представленных в базе данных.

AI-помощник предназначен исключительно для информационных целей и не заменяет консультацию врача.

---

## Поиск и нормализация услуг

Для повышения качества поиска используется словарь синонимов.

Например, следующие запросы считаются одинаковыми:

* ОАК;
* оак;
* Общий анализ крови;
* CBC;
* Complete Blood Count.

Все они приводятся к единому названию услуги, что позволяет корректно выполнять поиск независимо от формулировки запроса.

В текущей версии используются данные следующих ресурсов:

* KDL;
* Olymp;
* Invitro;
* Helix;
* DOQ;
* Medel;
* MCK;
* Aksai Clinic;
* другие открытые медицинские ресурсы, указанные в техническом задании.

Архитектура проекта позволяет добавлять новые источники без изменения основной логики приложения.
